# Ziqi (Jade) Qing - Personal Portfolio

A modern, responsive personal website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**, featuring a sophisticated **Morandi style** aesthetic.

## 🎨 Design Philosophy
- **Morandi Palette**: A curated selection of muted, desaturated tones including Rice-White (`#F5F3EF`), Blue-Grey (`#8FA3B0`), and Grey-Green (`#A3B18A`).
- **Minimalist Aesthetic**: Clean typography and subtle glassmorphism effects inspired by Vercel and Stripe.

## 🚀 Technical Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📂 Project Highlights
1. **Automated Regression Testing**: A robust E2E testing framework for data transformations using Cypress and GitHub Actions.
2. **NLP File Processor**: A full-stack Django/React system for processing files using natural language instructions.
3. **Competitive AI Agent**: A high-performance AI agent optimized through metric-driven experimentation.

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Zikki-Qing/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
